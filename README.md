# Procesamiento de Lenguaje Natural

Colección de notebooks prácticos desarrollados durante mi formación en Ciencia de Datos e Inteligencia Artificial.

Este repositorio documenta mi progresión en técnicas de procesamiento de lenguaje natural, desde extracción y preprocesamiento de texto hasta búsqueda semántica, recuperación de información y sistemas RAG.

## Contenidos

| Notebook | Tema |
|---|---|
| `01_pdf_text_processing.ipynb` | Extracción y procesamiento de texto desde documentos PDF |
| `02_web_scraping_preprocessing.ipynb` | Web scraping y preprocesamiento de texto |
| `03_semantic_similarity_spacy.ipynb` | Similitud semántica con spaCy |
| `04_retrieval_chatbot.ipynb` | Chatbot basado en recuperación de información |
| `05_rag_chatbot_embeddings.ipynb` | Sistema RAG con embeddings, Chroma y LLM |

## Progresión técnica

```text
Extracción de texto
        ↓
Preprocesamiento
        ↓
Representación semántica
        ↓
Recuperación de información
        ↓
Embeddings y base vectorial
        ↓
Retrieval-Augmented Generation
```

## Tecnologías

* Python
* Pandas
* spaCy
* Sentence Transformers
* Hugging Face Transformers
* Chroma
* PyTorch
* Jupyter Notebook

## Objetivo del repositorio

El objetivo de este repositorio es documentar la implementación práctica de técnicas de Procesamiento de Lenguaje Natural.

Los notebooks muestran una progresión desde técnicas tradicionales de procesamiento y similitud semántica hasta la implementación y evaluación de un sistema RAG utilizando embeddings, recuperación vectorial y modelos generativos.
